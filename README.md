# TG-Student 📚

A Flutter-based Student Management Application designed to help students easily access academic information such as attendance, assignments, notices, and study resources.

## 🚀 Features

* Student Login System
* View Attendance
* Check Assignments
* Notices & Announcements
* Study Resources
* Clean and Simple UI

## 🛠️ Technologies Used

* Flutter
* Dart
* Firebase (Authentication / Database)
* REST APIs

## 📂 Project Structure

```
lib/
 ├── screens/
 ├── widgets/
 ├── models/
 ├── services/
 └── main.dart
```

## 📦 Installation

1. Clone the repository

```
git clone https://github.com/ajimnadaf/TG-Student.git
```

2. Navigate to project folder

```
cd TG-Student
```

3. Install dependencies

```
flutter pub get
```

4. Run the app

```
flutter run
```
